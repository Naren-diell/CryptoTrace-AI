# CryptoTrace-AI

## AI-Powered Cryptocurrency Transaction Analysis & End Receiver Identification Platform

CryptoTrace-AI is a full-stack web application that helps analyze cryptocurrency transactions and identify the probable end receiver of funds using blockchain analysis and AI.

## Features

- Cryptocurrency transaction tracing
- Bitcoin & Ethereum support
- AI-powered investigation assistant
- Wallet risk analysis
- Interactive transaction graph
- Investigation reports

## Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MySQL
